# Curso Angular

Este repositório contém anotações e códigos produzidos no curso de [Angular](https://loiane.training/course/angular/).

## Requisitos
- [Node.JS](https://nodejs.org/en/).
- [NPM](https://www.npmjs.com/get-npm)
- TypeScript
`npm install -g typescript`

### Anotações
> Os comandos são realizados via terminal (Angular CLI).

Instalar Angular CLI:
`npm install -g angular-cli`

Criar projeto:
`ng new primeiro-projeto`

Subir Projeto:
`ng serve`

Criar Módulo:
`ng g m nome-modulo`

Criar Componente:
`ng g c nome-componente`

Criar Serviço:
`ng g s nome-service`

Criar Diretiva:
`ng g d nome-directive`

Criar Pipe:
`ng g p nome-pipe`

Criar Class:
`ng g c nome-classe`

Criar Interface:
`ng g i nome-interface`

Criar Enum:
`ng g e nome-enum`

Verificar boas práticas de código:
`ng lint`

Executar os testes:
`ng test`

`ng e2`
